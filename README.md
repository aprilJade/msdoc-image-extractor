# msdoc image extractor

마이크로소프트 문서에 포함되어있는 이미지 파일을 추출하는 프로그램입니다.

# 개발 환경
|분류|내용|
|---|---|
|OS|Windows 11|
|IDE|Visual Studio Community 2022|
|SDK Version|10.0.22621.0|
|Framework|MFC|


# 지원하는 파일 형식
xlsx, pptx, docx 등 xml기반 ms 문서
